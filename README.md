# Excel Sales Data Analysis

## Overview
This project demonstrates basic data analysis using Microsoft Excel.

## Dataset
The dataset includes:
- Date
- Product
- Region
- Sales Amount

## Tools Used
- Microsoft Excel
- Basic formulas (SUM)
- Data organization

## Objective
To practice organizing business data and performing simple sales analysis.

## Outcome
Generated a structured dataset and calculated total sales to understand product and regional sales performance.
